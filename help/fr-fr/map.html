Recherche de signaux
  • Filtres : Dans l'onglet Filtres, utilisez Ajouter (ex : Type IF, Code Ligne).
  • Légende : Cliquez sur une catégorie pour isoler les signaux correspondants.
  • ID Réseau : Filtrez par identifiant, puis cliquez sur la pastille pour zoomer sur le signal.
  • Code ligne : Filtrez par code ou libellé de ligne, puis cliquez sur la pastille pour zoomer sur l'emprise de la ligne. Le survol d'une pastille affiche son emprise géographique sous forme de contour pointillé avec un effet de mise en valeur — la prévisualisation disparaît dès que la carte est déplacée.
Interaction avec la carte
  • Contrôles : Clic droit sur un signal pour ouvrir le menu contextuel ou utilisez les raccourcis clavier ci-dessous.
  • Visibilité : Tous les signaux apparaissent à partir du zoom 10. En dessous, un échantillonnage spatial assure la lisibilité (les filtres restent prioritaires).
  • Persistance : Les filtres, signaux épinglés, position et réglages sont conservés entre les sessions.
Infobulle au survol

Survoler un marqueur affiche une infobulle. Chaque ligne indique le type de signal (libellé coloré) et son ID Réseau. Lorsque des signaux co-localisés diffèrent par leur sens ou leur position, ils sont regroupés en sous-blocs séparés par un trait. Le code ligne, le code voie, le nom de voie et le point kilométrique sont communs à tous les signaux co-localisés et apparaissent en bas.

Indicateurs de présence OSM, affichés dès que l'application a vérifié la zone :

IndicateurSignification

La vérification se déclenche automatiquement après 30 secondes d'immobilité à un zoom suffisant (≥ 14, moins de 100 signaux visibles), ou immédiatement au survol d'un marqueur.
